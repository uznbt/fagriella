# APK Android Native (Legacy Support)

Meskipun PWA adalah metode akses yang paling disarankan, kami tetap menyediakan versi **APK Android** bagi mahasiswa yang lebih menyukai aplikasi dalam format paket instalasi tradisional.

## Cara Menginstal APK

1.  Buka menu **Aplikasi & PWA** di sidebar Dokumentasi.
2.  Klik tautan **Download APK** (Jika tersedia di dashboard pengumuman).
3.  Buka file `.apk` yang telah diunduh di HP Anda.
4.  **Izinkan Sumber Tidak Dikenal**: Karena aplikasi ini didistribusikan secara mandiri (bukan lewat Play Store), Anda mungkin perlu memberikan izin ini di pengaturan HP.
5.  Klik **Instal** dan selesai.

---

## Perbedaan APK vs PWA

| Fitur | APK Android | PWA (Web) |
| :--- | :--- | :--- |
| **Ukuran** | ~2-5 MB | < 1 MB |
| **Instalasi** | Manual (.apk) | Langsung via Browser |
| **Pembaruan** | Harus download ulang | Otomatis (Auto-sync) |
| **Offline Mode** | Terbatas | Sangat Baik (Service Worker) |
| **Kompatibilitas** | Hanya Android | Semua (Android & iOS) |

---

> [!WARNING]
> **Rekomendasi Utama**: Kami sangat menyarankan penggunaan **Versi PWA** karena fitur pembaruan otomatisnya. Dengan PWA, Anda akan selalu mendapatkan perbaikan bug dan fitur materi terbaru tanpa perlu repot mengunduh file baru setiap minggu.

# Panduan: Instalasi & Akses PWA

F.AGRIELLA bukan hanya sekadar website, tapi juga sebuah **Progressive Web App (PWA)** yang bisa Anda instal di layar utama HP Anda dengan performa yang mirip aplikasi asli.

## Cara Menginstal PWA

### A. Perangkat Android (Google Chrome)
1.  Buka situs **fagriella.vercel.app** di Chrome.
2.  Tunggu banner "Tambahkan ke Layar Utama" muncul di bawah, atau...
3.  Ketuk ikon titik tiga (⋮) di pojok kanan atas.
4.  Pilih **Instal Aplikasi**.
5.  Konfirmasi pemasangan. Ikon F.AGRIELLA akan muncul di menu aplikasi Anda.

### B. Perangkat iOS / iPhone (Safari)
1.  Buka situs di Safari.
2.  Ketuk ikon **Share** (kotak dengan panah ke atas) di bagian bawah.
3.  Gulir ke bawah dan pilih **Add to Home Screen**.
4.  Beri nama dan klik **Add**.

---

## Keuntungan Versi PWA
1.  **Tanpa Menghabiskan Memori**: PWA sangat ringan (kurang dari 1MB), jauh lebih kecil dibandingkan aplikasi Play Store biasa.
2.  **Akses Instan**: Anda tidak perlu mengetik alamat URL setiap kali ingin mengecek tugas; cukup ketuk ikon di layar HP.
3.  **Splash Screen**: Tampilan awal (loading screen) premium saat aplikasi dibuka, memberikan kesan aplikasi profesional.
4.  **Notifikasi Push**: Memungkinkan sistem push notif bekerja lebih optimal di latar belakang.

> [!TIP]
> **Update Aplikasi**: Anda tidak perlu melakukan update manual. Setiap kali Anda membuka PWA saat online, sistem akan secara otomatis mengunduh versi terbaru dari Vercel di latar belakang.
